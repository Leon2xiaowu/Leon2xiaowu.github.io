微信H5获取iBeacons设备信息

· 2018-06-05 · # Javascript

前提条件

  1. 公众号申请开通摇一摇周边
  2. 添加设备并激活对应设备
  3. 拿到设备的一系列参数

设备信息类似于： 请将下表的UUID、Major、Minor信息配置至iBeacon设备，其中，major、minor为10进制。

部署设备 设备ID UUID major值 minor值
设备zero 3214551 FDA15795-A3E2-4FB1-DEWF-C6EB07154725 10100 44511

1、2是前提条件，不满足API不会有任何反应 3 是后续会用到的参数，让PM弄好了给你，如果你自己就是PM就在公众号里找找，可以导出对应Excel

获取beacon信息

H5页面获取设备信息 JS API

微信提供了开启、监听、关闭三个Beacon对应的接口。 在开始查找之前还需要将设备添加至对应分组，否则onSearchBeacons不会有任何反应、不进入任何回调

添加beacon至分组

根据文档-(新增分组)[https://mp.weixin.qq.com/wiki?t=resource/res_main&id=mp1459330208]

然后将步骤3中的设备加到分组中，onSearchBeacons只能监听位于分组中的beacon设备

监听beacons

监听周边beacon前需要执行startSearchBeacons开启查找beacon

接着在成功回调中执行onSearchBeacons监听周围的beacon设备

没反应的原因

在开发过程中遇到的没有反应的原因有以下几种

  1. 设备不在分组中
  2. 公众号没有添加设备
  3. 对应beacon设备没有绑定当前页面的URL
  4. 注册微信SDK的公众号和添加设备的公众号不一致

关闭beacon搜索

开启startSearchBeacons后，微信中关闭网页并不会自动stopSearchBeacons。 只能手动执行stopSearchBeacons关闭，可添加beforeunload listener在页面卸载前手动关闭
